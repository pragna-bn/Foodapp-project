# Django Food Ordering App

A full-stack Django web application for ordering food online from multiple restaurants. Users can browse restaurants, view menus, add items to cart, place orders, and manage favorites.

## Features
- Browse restaurants and their menus
- Add items to cart and checkout
- Place orders with customer details
- Mark food items as favorites
- Trending and popular items highlighted
- Admin panel to manage restaurants, food items, and orders
